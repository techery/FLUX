FLXStore

@interface FLXStore <FLXStateType> : NSObject<FLXDispatcherProtocol>

Base class for FLUX store. It’s designed to be subclassed. FLXStateType is a generic that defines a store’s state type

  • -onAction:callback:

    Subscribes store on specific action class with a callback. Each store subclass is responsible for subscribing to all actions it is interested in on initialization.

    Declaration

    Objective-C

    Swift

    Parameters

    class of the action which will be processed by current store
    callback

    action block that defines how state will be changed
